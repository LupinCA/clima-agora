# 🌤️ Clima Agora

Aplicação de clima responsiva desenvolvida com HTML, CSS e JavaScript utilizando a API OpenWeatherMap.

O projeto permite pesquisar cidades em tempo real e exibir informações climáticas de forma dinâmica e visualmente moderna.

---

## ✨ Funcionalidades

* 🔎 Busca de cidades em tempo real
* 🌡️ Temperatura atual
* 🤝 Sensação térmica
* 💧 Umidade do ar
* 🌬️ Velocidade do vento
* 🌙 Tema dinâmico para dia/noite
* ☀️ Alteração visual conforme clima
* ⏳ Loading spinner animado
* 📱 Layout responsivo para mobile

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* OpenWeatherMap API

---

## 🎨 Conceitos praticados

* Consumo de API com Fetch
* Async/Await
* Manipulação do DOM
* Responsividade
* Glassmorphism
* Animações com CSS
* Organização de código
* Separação de responsabilidades
